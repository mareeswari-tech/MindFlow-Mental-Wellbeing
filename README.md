# MindFlow — Design Thinking for Mental Well-Being

## Project Overview
A student mental well-being mobile application 
built using Design Thinking methodology.

Subject: NM 1075 EBPL-Design Thinking  
Project Code: DTP008  
Team: SDC1208  
Created By: A. Mareeswari  

## Tech Stack
- HTML5 — Structure & Layout
- CSS3 — Responsive Design & Animations  
- JavaScript — Logic & Interactivity
- React — Component Architecture & State Management

## Key Features
- Mood Tracker & Journaling
- Breathing & Mindfulness Exercises
- Crisis Support (SOS Button)
- Resource Hub
- Peer Community Support
- Progress Tracking Dashboard

## Design Thinking Process
1. Empathize — Student interviews & empathy mapping
2. Define — Problem statement
3. Ideate — Brainstorming solutions
4. Prototype — Working UI
5. Test — 3 rounds of user testing
